Class: Headers

lib/headers.Headers

new Headers(channel, direction)

Headers is an object to access and modify headers of a request. It implements an interface similar to the JS Map object. It also implements the iterable protocol, so you can iterate over it with for...of loops. Each value when iterating is a module:lib/headers~HeaderTuple.
