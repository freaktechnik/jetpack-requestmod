Class: Headers

Headers

new Headers(channel, direction)

Headers is an object to access and modify headers of a request. It implements an interface similar to the JS `Map` object. It also implements the iterable protocol, so you can iterate over it with for...of loops. Each value when iterating is an object with a property `header` storing the name of the header and `value` containing the header's value.
